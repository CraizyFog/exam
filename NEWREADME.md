# Патерни проєктування (TypeScript)

Реалізація популярних патернів проєктування (GoF) мовою TypeScript.

## Структура проєкту

- `src/creational/` — породжувальні патерни (Фабричний метод).
- `examples/` — скрипт для запуску та тестування патернів.

## Налаштування та запуск

1. Встановіть залежності:
   ```bash
   npm install


     npx ts-node examples/run-factory-method.ts
